# PSLab Application

The PSLab application provides an easy-to-use interface for interacting with
Pocket Science Lab hardware. It allows users to perform experiments, collect
data, and access PSLab features across multiple platforms.

## Available Applications

| Platform | Availability |
|----------|--------------|
| Android | Available on [Google Play Store](https://play.google.com/store/apps/details?id=io.pslab) and [F-Droid](https://f-droid.org/de/packages/io.pslab/) |
| iOS | Available on the [App Store](https://apps.apple.com/app/pslab/id6740454978) |
| Windows | Development builds available through [GitHub](https://github.com/fossasia/pslab-app/tree/app) |
| macOS | Development builds available through [GitHub](https://github.com/fossasia/pslab-app/tree/app) |
| Linux | Development builds available through [GitHub](https://github.com/fossasia/pslab-app/tree/app) |

## Installation

A Flatpak build of the application is available on
[Flathub](https://flathub.org/apps/io.pslab.PSLab).

## Windows Driver Requirement

When using the PSLab application on Windows, the CP2102N USB serial adapter driver may be required for communication with the device.

Install the driver from the [Silicon Labs website](https://www.silabs.com/software-and-tools/usb-to-uart-bridge-vcp-drivers).
